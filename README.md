# 이젠 아카데미 5조 프로젝트 2번째 📚


## 🎈한/영 단어장 프로그램 제작
  JavaFX와 DB 접근 처리에 대한 가장 기반이 되는 기술인 JDBC 학습을 위해 진행한 한/영 단어장 프로젝트입니다.
   
   사용자가 단어를 추가, 수정, 검색, 삭제 할 수 있도록 개발한 프로그램 입니다.
